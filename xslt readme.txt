Below are the files which are related to XSLT report:
1. build.xml
2. testng-results.xsl
3. lib\sason-license\
4. lib\saxon-8.7.jar
5. lib\saxonliaison.jar

How to generate XSLT repoort?
Ant build is required to generate this report. First need to install ANT and once it is installed go to commandprompt and to the project home directory where build.xml file exits and then enter ant and hit enter.
This will generate index.html file in test-output folder as the directory location and file name mentioned in the build.xml file.
